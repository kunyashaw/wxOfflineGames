# 微信小游戏小程序

> 50款免费单机小游戏合集

## 项目介绍

一款集合5大类、50款免费单机小游戏的微信小程序，包含益智烧脑、经典街机、休闲消除、棋类策略、三消消除等多种类型游戏。

## 功能特性

- 📚 5个游戏类别，每类10款游戏
- 💾 独立存档系统
- 🏆 积分排行榜
- ❤️ 收藏功能
- 🎮 金手指作弊
- 🔊 音效系统
- 📤 分享功能

## 游戏列表

### 益智烧脑类 (10款)
2048、数独、拼图、记忆棋、推箱子、华容道、数字连线、颜色匹配、管道工、密码破译

### 经典街机类 (10款)
贪吃蛇、俄罗斯方块、打砖块、赛车跑酷、弹珠台、直升机、飞机大战、采金矿、砖块跳跃、冰块滑行

### 休闲消除类 (10款)
扫雷、接龙、对对碰、麻将消消、点消大师、切割大师、水果连线、找不同、俄罗斯消除、井字棋

### 棋类策略类 (10款)
五子棋、黑白棋、国际象棋、迷宫、跳棋、中国象棋、围棋入门、围捕、策略塔防、2048策略版

### 三消消除类 (10款)
宝石迷阵、泡泡龙、糖果消消乐、祖玛、水果忍者、冰雪消消、动物三消、饼干消消、宝石交换、农场三消

## 技术栈

- 微信小程序原生开发
- Canvas 2D 游戏渲染
- 本地存储

## 项目结构

```
miniprogram/
├── app.js              # 应用入口
├── app.json            # 应用配置
├── app.wxss           # 全局样式
├── utils/             # 工具函数
│   ├── gameConfig.js   # 游戏配置
│   ├── storage.js      # 本地存储
│   └── audio.js        # 音效管理
├── pages/              # 页面
│   ├── home/           # 首页
│   ├── category/       # 分类页
│   ├── game/           # 游戏页
│   └── profile/        # 我的
└── games/              # 游戏模块
    ├── base/           # 游戏基类
    ├── puzzle/         # 益智类
    ├── arcade/         # 街机类
    ├── casual/         # 休闲类
    ├── chess/          # 棋类
    └── match3/         # 三消类
```
